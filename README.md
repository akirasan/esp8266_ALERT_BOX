# esp8266 ALERT BOX
Sistema de alerta visual y controlado remotamente.

## Requisitos:
- Servidor MQTT (Mosquitto, por ejemplo)
- Servidor Node-RED (es opcional, pero ideal para controlar mensajes por MQTT)
